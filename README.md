# 🫁 Lung Cancer Risk Screening App

An AI-powered mobile application that predicts lung cancer risk based on user input and provides health insights.

---

## 🚀 Features

- 📊 Risk Assessment (Low / Medium / High)
- 🤖 AI Chatbot
- 🧑‍⚕️ Doctor Finder
- 📄 PDF Health Report
- 🔐 JWT Authentication

---

## 🛠 Tech Stack

- React Native
- Spring Boot
- Axios
- AsyncStorage
